  • Cyber Security

    Facial Intruder Alert System

    The Facial Intruder Alert System is a cybersecurity-powered security solution that continuously monitors personal devices using real-time facial recognition. It detects unauthorized access attempts and issues immediate alerts through pop-ups, phone calls, and email notifications. The system can lock the device, track multiple faces, and respond to suspicious behavior such as camera obstruction. Designed to operate independently of OS-level security, it offers robust protection in shared or public environments.

  • RAG (Retrieval Augmented Generation)

    RAG AI

    RAG AI is an intelligent, retrieval-augmented system that ingests content from uploaded PDFs or website links and delivers accurate, context-aware answers based on user queries. It combines document-based search with generative AI capabilities to ensure high relevance and precision in responses. Users can switch to a general AI chat mode for open-ended conversations, similar to ChatGPT functionality. Designed for flexibility and efficiency, RAG AI enhances knowledge access and delivers dynamic, real-time insights.

  • Machine Learning

    AI powered Vulnerability Scanner

    The AI-powered Vulnerability Scanner is an intelligent security analysis system that combines pattern-based techniques with machine learning to detect complex and context-specific code flaws. It performs static code analysis using methods like regex matching, AST parsing, and control flow inspection for comprehensive vulnerability detection. Machine learning models enhance accuracy by classifying threats and reducing false positives. Designed for seamless integration into development workflows, the system offers automated remediation, detailed risk reports, and real-time feedback to strengthen software security throughout the development lifecycle.
